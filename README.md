# Cheatsheet Generator

A React-based application for creating and exporting cheatsheets as PDF documents using Markdown.

<img width="1875" height="1014" alt="image" src="https://github.com/user-attachments/assets/41151eb6-f043-46bf-9ac5-9b07d5018510" />



## Features

- Real-time Markdown editing with syntax highlighting
- Live preview of formatted Markdown content
- PDF export via browser print functionality
- Customizable multi-column layout settings
- Mathematical expressions with KaTeX
- Diagrams with Mermaid
- Automatic saving using localStorage

## Tech Stack

- React
- Vite
- Monaco Editor for code editing
- KaTeX for mathematical expressions


## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm run dev
   ```

3. Build for production:
   ```
   npm run build
   ```
